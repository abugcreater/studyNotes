# 多租户简介

## 概念

**多租户技术**（英语：**multi-tenancy technology**）或称**多重租赁技术**，是一种[软件架构](https://zh.wikipedia.org/wiki/軟體架構)技术，它是在探讨与实现如何于多用户的环境下共享相同的系统或程序组件，并且仍可确保各用户间资料的隔离性。
